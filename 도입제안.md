# UTM Guard 도입 제안 — 1장 요약

## 한 줄

> 넘버스는 **모든 셀이 자유 입력, 모든 검증이 사람 눈**이라 오류가 항상 *배포된 뒤 GA에서* 발견됩니다.
> UTM Guard는 오류 6종 중 **5종을 발생 자체가 불가능**하게, 나머지 **1종을 붙여넣는 즉시 자동 감지**합니다.
> → "사고를 줄이는" 게 아니라 **"사고 가능 지점을 없애는"** 도구입니다.

---

## 우리가 실제로 겪던 문제 (현 캠페인 시트 기준)

- 대소문자 드리프트 — `Ig / ig / instagram` 이 GA에서 3개 채널로 집계
- 캠페인 오타 — `roadshow-busan` vs `roadshow-busman` 이 한 캠페인을 둘로 쪼갬
- `utm_content` 중복 — 같은 캠페인 안에서 행을 구분 못 함
- 깨진 URL — `https//`(콜론 누락), 스킴 없음, `fasttrack2026//postech`
- 빈 `utm_*` — 클릭이 `(not set)`으로 샘
- Whole-URL ↔ 컴포넌트 불일치 — 손으로 친 두 사본이 서로 어긋남

진짜 비용은 타이핑 시간이 아니라 **리포트가 조용히 쪼개지는 것**, 그리고 몇 주 뒤
"왜 숫자가 안 맞지?"를 역추적하는 시간 — 보이지 않고 상한도 없음.

## 넘버스 vs UTM Guard

| 오류 유형 | 넘버스 | UTM Guard | 보장 |
|---|---|---|---|
| 대소문자 드리프트 | 무방비 | 선택지에서만 고름 + 소문자 강제 | 🔒 원천 차단 |
| 캠페인 오타 | 눈으로 확인 | 기존 값 선택 (+오타 자동 감지) | 🔒 차단+감지 |
| URL ↔ 컴포넌트 불일치 | 손으로 친 URL이 따로 놀음 | URL을 컴포넌트로부터 **생성** | 🔒 원천 차단 |
| 깨진 URL | 무방비 | URL 안 침 + 스킴/이중슬래시 검사 | 🔒 차단+감지 |
| 빈 `utm_*` | 발사 후 발견 | 에러 표시 + **에러 시 복사 비활성** | 🛡 발사 차단 |
| `utm_content` 중복 | 사실상 안 보임 | **자동 채번**(캠페인별 다음 빈 번호) + 중복 시 복사 차단 | 🔒 원천 차단 |

## 숫자로

- **자동 검사**: 넘버스 0회(전부 수동) → 붙여넣는 **모든 행 × 8종 검사** 자동
- **발견 시점**: "몇 주 뒤 GA에서 분석가가 의아해할 때" → **복사 직전**
- **불가능해지는 오류**: 6종 중 **5종**, 나머지 1종은 100% 자동 감지
- **전환 비용 0**: 쓰던 넘버스 시트를 복사-붙여넣기/CSV로 가져오면 실제 쓰던 값이 그대로 선택지가 됨

---

## 도입 방법 (전환 비용 거의 없음)

1. 쓰던 시트를 **Lists 탭**에 붙여넣기(또는 CSV 열기) → 선택지 자동 등록
2. 한 명이 **선택지 내보내기(.json)** → 공유 드라이브/리포지토리에 두기
3. 팀원은 **공유 목록 가져오기** → 전원이 동일한 표준 선택지 사용
4. 새 링크는 **Builder 탭**에서 드롭다운으로 생성 → 복사
5. 기존 시트는 **Sheet check 탭**에 붙여넣어 한 번에 감사

요구 사항: macOS 14+, 의존성·서명 없음, 로컬 실행.

---

## 팀 표준화 — 실시간 동기화 (신규)

- 공유 `.json` 파일에 **연결**하면 변경이 **자동 반영**(2초 간격, 양방향).
  한 명이 새 캠페인을 추가하면 팀 전원의 드롭다운에 곧바로 나타남.
- "사람마다 표기가 다른" 근본 원인을 **조직 차원에서, 실시간으로** 제거.
- 서버·계정 불필요. Dropbox·Google Drive·Git 폴더에 파일 하나만 두면 됨
  (Git에 두면 변경 이력까지 남음). 앱 재시작 후에도 연결 자동 유지.
- 선택지는 합집합으로만 병합되어 **충돌 없이 항상 수렴**.

## 정직한 한계 (지금)

- 동기화는 **2초 폴링** 기반 — 즉각(밀리초)은 아니지만 실사용엔 충분
- 병합은 **합집합** — 한쪽에서의 삭제는 전파되지 않음 (잘못된 값 정리는 공유 파일 직접 수정)
- 넘버스 **실시간 연동 아님** — 붙여넣기/CSV 가져오기 방식
- 자동 채번은 **숫자(01, 02…)** 기준 — 다른 명명 규칙은 수동 입력(중복은 여전히 차단)

→ 다음 단계 후보: 삭제 전파(소프트 삭제), 채번 형식 커스터마이즈.
